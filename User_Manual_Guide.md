# ExamPortal User Manual Guide

## Getting Started

### Account Registration
1. Visit the ExamPortal login page
2. Click on the **Register** tab
3. Fill in your details:
   - Full Name
   - Email Address
   - Password (minimum 6 characters)
4. Click **Create Account**
5. You will be automatically logged in as a Student

### Login Process
1. Enter your registered email and password
2. Click **Sign In**
3. You will be redirected to your dashboard based on your role

### Forgot Password
1. Click **Forgot Password?** on the login page
2. Enter your email address
3. Click **Send Reset Link**
4. Check your email for password reset instructions
5. Follow the link to create a new password

## Student Guide

### Dashboard Overview
After logging in, you will see:
- Available exams list
- Your exam results
- Performance statistics
- Leaderboard access

### Taking an Exam

#### Starting an Exam
1. Find the exam in your dashboard
2. Check the exam details (duration, questions, start time)
3. Click **Start Exam** when ready
4. Confirm you want to begin in the popup
5. The exam will automatically enter fullscreen mode

#### During the Exam
- **Navigation**: Use the question grid on the left to jump between questions
- **Timer**: Monitor remaining time in the top-right corner
- **Answering Questions**:
  - **Multiple Choice**: Click on your chosen answer
  - **True/False**: Select True or False
  - **Short Answer**: Type your response in the text box
- **Question Status**:
  - Gray circle: Unanswered
  - Green circle: Answered
  - Blue circle: Current question

#### Important Exam Rules
- **Stay in fullscreen**: Do not exit fullscreen mode
- **No tab switching**: Avoid switching to other browser tabs
- **Time limits**: Submit before time expires
- **Auto-submit**: Exam submits automatically when time ends

#### Submitting Your Exam
1. Review your answers using the question navigation
2. Click **Submit Exam** when finished
3. Confirm submission in the popup
4. View your results immediately (for auto-graded questions)

### Viewing Results
1. Go to **All Results** section in your dashboard
2. Click on any exam to see detailed results
3. Review your score breakdown by question type
4. See correct answers for multiple choice and true/false questions

### Leaderboard
1. Access the **Leaderboard** from your dashboard
2. Filter by specific exam or subject
3. Compare your performance with other students
4. View rankings and scores

### Profile Management
1. Click on **Profile** in the sidebar
2. Update your personal information
3. View your exam statistics
4. Click **Save Changes** to update

## Teacher Guide

### Dashboard Overview
Your teacher dashboard includes:
- Exam management tools
- Student performance analytics
- Question bank management
- Reporting features

### Creating an Exam

#### Basic Exam Setup
1. Click **Create Exam** in your dashboard
2. Fill in exam details:
   - **Exam Title**: Name of the exam
   - **Subject**: Select or type subject
   - **Duration**: Time limit in minutes
   - **Start Date & Time**: When exam becomes available
   - **End Date & Time**: When exam closes
   - **Instructions**: Guidelines for students

#### Adding Questions
1. After creating exam details, add questions using floating buttons:
   - **Add MCQ**: Multiple choice questions
   - **Add True/False**: Boolean questions
   - **Add Short Answer**: Text response questions

2. **For Multiple Choice Questions**:
   - Enter question text
   - Add 4 answer options (A, B, C, D)
   - Select the correct answer
   - Set marks for the question

3. **For True/False Questions**:
   - Enter question text
   - Select correct answer (True or False)
   - Set marks for the question

4. **For Short Answer Questions**:
   - Enter question text
   - Provide sample answer for grading reference
   - Set maximum marks

#### Publishing the Exam
1. Review all questions and exam details
2. Click **Create Exam** to publish
3. Students can now see and take the exam during scheduled time

### Managing Questions

#### Question Bank Access
1. Go to **Add Questions** from the sidebar
2. View all existing questions by type
3. Search questions using the search box
4. Filter by question type (MCQ, True/False, Short Answer)

#### Adding New Questions
1. Click the **+** floating button
2. Select question type from the menu
3. Fill in question details
4. Click **Add Question** to save

#### Editing Questions
1. Find the question in the question bank
2. Click **Edit** button
3. Modify question details in the popup
4. Click **Update Question** to save changes

### Grading Short Answer Questions

#### Accessing Ungraded Questions
1. Go to **My Exams** section
2. Find exams with short answer questions
3. Click **Grade** button for exams needing manual grading

#### Grading Process
1. Review student's answer
2. Compare with sample answer
3. Enter marks (can use decimals like 1.5/2)
4. Click **Save Grade**
5. Repeat for all students and questions

### Student Management

#### Viewing Students
1. Access **Students** section from sidebar
2. View all registered students
3. Use search to find specific students
4. Filter by performance or exam participation

#### Student Performance Analysis
1. Click on any student name
2. View detailed performance metrics
3. See exam history and scores
4. Monitor progress over time

### Generating Reports

#### PDF Reports
1. Go to exam details or student management
2. Click **Generate PDF Report**
3. Report includes:
   - Exam statistics
   - Student performance
   - Question analysis
   - Grade distribution

#### Excel Reports
1. Click **Export to Excel** in relevant sections
2. Download comprehensive data including:
   - Student scores
   - Question-wise performance
   - Time analysis
   - Detailed breakdowns

### Leaderboard Management
1. Access **Leaderboard** from sidebar
2. Filter by exam or subject
3. View student rankings
4. Monitor class performance trends

## Troubleshooting

### Common Issues

#### Cannot Start Exam
- Check if exam is within scheduled time
- Ensure you haven't already completed the exam
- Verify your internet connection
- Try refreshing the page

#### Fullscreen Problems
- Allow fullscreen permission in your browser
- Use supported browsers (Chrome, Firefox, Safari, Edge)
- Disable browser extensions that might interfere

#### Questions Not Loading
- Check internet connection
- Refresh the page
- Clear browser cache
- Try a different browser

#### Cannot Submit Exam
- Ensure all required questions are answered
- Check if time hasn't expired
- Try submitting individual sections
- Contact your teacher if issues persist

### Browser Requirements
- **Recommended**: Chrome 80+, Firefox 75+, Safari 13+, Edge 80+
- **JavaScript**: Must be enabled
- **Cookies**: Must be allowed
- **Fullscreen**: Must be supported and allowed

### Getting Help
- Contact your teacher for exam-related issues
- Check your internet connection for technical problems
- Clear browser cache if pages aren't loading properly
- Use supported browsers for best experience

## Security and Fair Use

### For Students
- Take exams in a quiet, private environment
- Do not share login credentials
- Do not attempt to cheat or use unauthorized materials
- Report technical issues immediately
- Complete exams within designated time frames

### For Teachers
- Regularly monitor exam sessions
- Review suspicious activity logs
- Ensure fair grading practices
- Maintain student privacy and data security
- Backup important exam data regularly
