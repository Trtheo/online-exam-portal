# Programming Fundamentals - True/False Exam

## Instructions
Read each statement carefully and determine whether it is **True** or **False**.

---

## Question 1
**Statement:** A variable is used to store data in a program.

**Answer:** True

**Explanation:** Variables act as containers that hold values which a program can use or modify during execution.

---

## Question 2
**Statement:** In most programming languages, the symbol `=` is used to compare two values.

**Answer:** False

**Explanation:** The `=` symbol is usually for assignment (storing a value), while comparison uses `==` or another operator.

---

## Question 3
**Statement:** A loop allows a set of instructions to run repeatedly.

**Answer:** True

**Explanation:** Loops (like `for` and `while`) repeatedly execute code until a condition is met.

---

## Question 4
**Statement:** Syntax errors occur when a program runs but produces the wrong output.

**Answer:** False

**Explanation:** Syntax errors happen when code breaks language rules, preventing the program from running at all.

---

## Question 5
**Statement:** In Python, code blocks are defined by indentation.

**Answer:** True

**Explanation:** Python uses indentation (spaces or tabs) to group statements, rather than braces `{}`.

---

## Answer Key
1. True
2. False
3. True
4. False
5. True
